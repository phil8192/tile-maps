## Running

Generate model. Output in CPLEX LP format:

```
python3 ./model.py
```

Now feed it to an IP solver. Default is GNU Linear Programming Kit.
COIN-OR will be faster. Gurobi (propriatory will be fastest).

```
glpsol --lp tiles.lp
```
